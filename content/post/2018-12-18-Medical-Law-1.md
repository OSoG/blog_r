###1 진료거부 금지 의무 및 응급의료 의무
진료나 조산 요청을 받으면 정당한 사유 없이 거부할 수 없다.

```
정당한 사유

1. 의사가 부재 중이거나 신병으로 인하여 진료를 할 수 없는 경우
2. 환자가 의료인에 대하여 모욕죄, 명예훼손죄, 폭행죄, 업무방해죄에 해당하는 상황을 야기해 의료인이 정상적인 의료행위를 하지 못할 때
3. 진료 예상대기시간을 알려주어 환자가 다른 의료기관으로 갈 수 있도록 한 경우
4. 1차 시술을 받은 다른 의료기관에서 계속 진료하라고 권유한 경우
5. 한밤중에 응급의료기관이 아닌 곳에 내원한 환자를 인근 종합병원으로 안내한 경우

```

```
구체적 사례들

1. 진료시설이나 진료과목이 없어 다른 의료기관으로 이송하는 경우
2. 일단 진료한 환자의 상태를 보아 의사가 의학적인 판단에 따라 퇴원 또는 타의료기관을 권유
3. 치료,수술이 불가능하고 오히려 환자에게 불이익을 초래할 위험이 있다고 판단되는 경우
4. 의사의 부재(휴진 등) 또는 건강상 이유로 진료할 수 없는 경우
5. 입원실(중환자실, 수술실) 이 꽉 차서 입원치료가 불가능 한 경우
```

```
부적절한 사례(진료 거부에 해당. 진료해야해!)

1. 의료기관 종하자와 다툼이 있는 환자를 거부하는 경우
2. 어린아이가 진료 중 의사의 말을 따르지 않는다고 내쫓는 경우
3. 단순히 피곤하거나 개인적 감정으로 진료를 거부하는 경우
4. 연대보증을 하지 않아 진료를 거부하는 경우
5. 진료비 청구가 곤란하고 의사의 지시를 무시한 환자가 심한 요통으로 내원하여 진료를 거부한 경우
```

```
구체적 사례들
1. 진료비 청구가 곤란한 경우(연대보증을 하지 않은 경우)
2. 진료시간 외에 응급환자에 대한 진료를 거부하는 경우
3. 부속의료기관 주변에 의료기관이 없는 상황에서 응급환자에 대한 진료를 거부하는 경우
4. 단순히 피곤하거나 환자에 대한 개인적인 감정이 있는 경우

```
